# PCC — บริษัท พรีไซซ คอร์ปอเรชั่น จำกัด (มหาชน)

## ข้อมูลทั่วไป
- **ชื่อเต็ม:** บริษัท พรีไซซ คอร์ปอเรชั่น จำกัด (มหาชน)
- **ชื่อย่อ:** PCC
- **เว็บไซต์:** https://precise.co.th
- **ข้อมูลกรรมการ:** https://precise.co.th/about/board-of-directors-and-committees/

## โครงสร้างกรรมการ (Board Structure)
- กรรมการทั้งหมด: **12 คน**
- กรรมการอิสระ: **6 คน** (50%) — เกณฑ์ต้องการ >50%
- กรรมการหญิง: **3 คน** (25%) — เกณฑ์ bonus ≥30%
- กรรมการไม่เป็นผู้บริหาร: **8 คน** (67%) — เกณฑ์ต้องการ >66%
- อายุงานสูงสุด: 9 ปี

## คณะกรรมการชุดย่อย (Sub-committees)
| คณะ | ประธาน | จำนวนสมาชิก |
|-----|--------|------------|
| AC (ตรวจสอบ) | สุดารัตน์ ตั้งสุนทรกิจ | 3 คน |
| NRC (สรรหาฯ) | รศ.ดร.ภก.ลือรัตน์ อนุรัตน์พานิช | 3 คน |
| RMC (ความเสี่ยง) | กิตติ ณัฐชยางกุล | 3 คน |
| GCG (กำกับดูแล) | สรรค์ชัย เตียวประเสริฐกุล | 3 คน |
| ExCo (บริหาร) | กิตติ สัมฤทธิ์ | 4 คน |
