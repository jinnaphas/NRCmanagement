## บทบาทเลขา NRC (Earth)
- **เจ้าของบทบาท:** จิณนภัสฐ์ ภัสฐาพงษ์ (Earth)

- **รับช่วงจาก:** เลขาคนเดิม (เอกสารกำลังย้าย Folder)

## หน้าที่หลักของเลขา NRC
1. จัดส่งหนังสือเชิญประชุมและระเบียบวาระ
2. จดรายงานการประชุม (MOM)
3. ติดตามมติที่ประชุม
4. จัดเตรียมเอกสารประกอบวาระ
5. ดูแล Control List ประจำปี
6. บริหาร Succession Plan กรรมการ
7. สนับสนุนข้อมูล CG / Board Skills Matrix

## กระบวนการบริหารด้วยแอป (v3 — สรุป)

- รูป/ไฟล์ = Google Drive share link · ไม่ต้องล็อกอิน · CG/Skills Matrix คิดจากสมาชิกบอร์ด

## เกณฑ์ CG ที่ NRC ดูแล (วัดจาก Board ทั้งหมด)
| เกณฑ์ | ค่าที่ต้องการ | สถานะปัจจุบัน |
|------|-------------|--------------|
| กรรมการอิสระ | >50% | 50% (6/12) — ผ่านพอดี |
| กรรมการหญิง | ≥30% (bonus) | 25% (3/12) |
| กรรมการไม่เป็นผู้บริหาร | >66% | 67% (8/12) — ผ่าน |
| อายุงานสูงสุด | ≤9 ปี | ตรวจสอบรายคน |
