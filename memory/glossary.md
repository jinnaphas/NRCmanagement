# Glossary — NRC / PCC

Workplace shorthand, acronyms, and internal language.

## คณะกรรมการชุดย่อย (Sub-committees)
| คำย่อ | ชื่อเต็ม (ไทย) | ชื่อเต็ม (อังกฤษ) |
|------|--------------|-----------------|
| NRC | คณะกรรมการสรรหาและพิจารณาค่าตอบแทน | Nomination and Remuneration Committee |
| AC | คณะกรรมการตรวจสอบ | Audit Committee |
| RMC | คณะกรรมการบริหารความเสี่ยง | Risk Management Committee |
| GCG | คณะกรรมการกำกับดูแลกิจการ | Good Corporate Governance Committee |
| ExCo | คณะกรรมการบริหาร | Executive Committee |

## องค์กรและบริษัท
| คำย่อ | ความหมาย |
|------|---------|
| PCC | บริษัท พรีไซซ คอร์ปอเรชั่น จำกัด (มหาชน) |
| บมจ. | บริษัทมหาชนจำกัด |
| บจ. | บริษัทจำกัด (บริษัทย่อย) |

## คำศัพท์ CG / กรรมการ
| คำ | ความหมาย |
|----|---------|
| CG | Corporate Governance (การกำกับดูแลกิจการ) |
| กรรมการอิสระ | Independent Director — ต้องมีสัดส่วน >50% ของบอร์ดทั้งหมด |
| กรรมการบริหาร | Executive Director |
| กรรมการไม่เป็นผู้บริหาร | Non-Executive Director |
| ประธาน | Chairman |
| รองประธาน | Vice Chairman |
| MOM | Minutes of Meeting (รายงานการประชุม) |
| วาระ | Agenda Item (วาระการประชุม) |
| กฏบัตร | Charter (กฏบัตรของคณะกรรมการ) |

## วันที่และปี
| คำ | ความหมาย |
|----|---------|
| พ.ศ. / BE | ปีพุทธศักราช = ปี ค.ศ. + 543 |
| ค.ศ. / CE | ปีคริสต์ศักราช |
| 2569 | พ.ศ. 2569 = ค.ศ. 2026 |
| 2568 | พ.ศ. 2568 = ค.ศ. 2025 |
| 2566 | พ.ศ. 2566 = ค.ศ. 2023 |

## AI / Strategy
| คำ | ความหมาย |
|----|---------|
| Agentic AI | AI ที่ทำงานเองเป็นขั้นตอน, วางแผนและตัดสินใจได้ |
| Agentic AI Organization | องค์กรที่ใช้ Agentic AI ในกระบวนการหลัก |
| Transform | การเปลี่ยนแปลงองค์กรเชิงกลยุทธ์ |

## เอกสารและงาน NRC
| คำ | ความหมาย |
|----|---------|
| หนังสือเชิญประชุม | Meeting Invitation Letter |
| ระเบียบวาระ | Meeting Agenda |
| รายงานการประชุม | Minutes of Meeting (MOM) |
| Control List | รายการควบคุมงาน NRC ประจำปี |
| Succession Plan | แผนสืบทอดตำแหน่ง |
| Board Skills Matrix | ตารางทักษะกรรมการ |
| Scorecard | คะแนนวัดคุณภาพ (เช่น CG Scorecard) |
